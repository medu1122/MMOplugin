# Thư mục đặt icon (ảnh PNG)

**Bỏ các file ảnh icon vào đây.**

## Tên file cần có

| Tên file | Dùng cho |
|----------|----------|
| `icon_role_dps.png` | Role DPS |
| `icon_role_tanker.png` | Role Tanker |
| `icon_role_healer.png` | Role Healer |
| `icon_role_logo.png` | Logo / chữ "ROLE" |
| `icon_btn_change.png` | Nút đổi role |
| `icon_btn_skills.png` | Nút skills (icon Fireball – cùng icon_skill_fireball) |
| `icon_btn_select_skill.png` | Nút chọn skill |
| `icon_btn_titles.png` | Nút danh hiệu |
| `icon_btn_close.png` | Nút đóng |
| `icon_skill_fireball.png` | Skill Fireball |
| `icon_skill_<id>.png` | Skill khác (thay \<id\> bằng ID skill) |

## Yêu cầu ảnh

- **Định dạng:** PNG, nền trong suốt.
- **Kích thước:** 16×16, 32×32 hoặc 64×64 (khuyến nghị 32×32).
- Mỗi file một icon; plugin sẽ map qua CustomModelData.
